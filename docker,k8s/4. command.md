# docker의 명령어 형태
docker의 기본적인 명령어 형태는 다음과 같다.
docker 커맨드 (옵션) 대상 (인자)
커맨드는 상위 커맨드와 하위 커맨드로 나뉘고 상위 커맨드는 '무엇을' 하위 커맨드는 '어떻게' 할것인지 나타낸다.
커맨드에 container를 생략할 수 있는 명령어들이 있는데, 관례상 생략하고 작성하는 편이 많다.
